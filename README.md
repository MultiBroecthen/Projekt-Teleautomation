# Projekt-Teleautomation - Smartwatches in der AT

Studentisches Projekt im WS 17/18 zur Entwicklung einer Android Wear App zur Überwachung und Steuerung der Kleinversuchsanlage der AgTele des IfA der TU Dresden

Die Kommunikation mit dem Server funktioniert nur aus dem Uni-Netz.

Optimiert für Smartwatches mit rechteckigem Display.

Getestet mit Sony Smartwatch 3.
